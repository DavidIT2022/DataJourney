Per visualizzare la presentazione:

OPZIONE 1 - Semplice:
Apri il file "index.html" con un browser moderno
(Chrome, Firefox, Edge, Safari)

OPZIONE 2 - Server locale (per funzionalità complete):
Se hai Python installato, apri il terminale in questa cartella e scrivi:
   python -m http.server 8000
Poi apri il browser su: http://localhost:8000

CONTENUTI:
- index.html  → Pagina principale
- style.css   → Stili grafici
- script.js   → Animazioni e interattività

NAVIGAZIONE:
- Usa il menu hamburger in alto a destra
- Scorri con il mouse o le frecce ↑ ↓
- Su mobile: swipe per aprire il menu
